ug nx 8.5 的安装

发表于 2019-09-04 | 分类于 记录 | 评论数： | 阅读次数：
本文字数： 1.8k | 阅读时长 ≈ 2 分钟

这学期要学 ug nx 8.5 ，记录一下这个沙雕软件的超级繁琐安装破解过程，安完之后又一次感叹：有钱还是用正版吧，盗版软件真的麻烦……

声明：本文只讨论学习盗版 ug nx 8.5 的安装过程，安装后即卸载，坚决抵制使用盗版软件！

首先关于安装包，网上的破解安装包基本上都是一个思路，跟着我的步骤走应该都行得通，没必要重新下载和我一样的安装包，如果下载了一样的安装包还行不通，建议换电脑或者换脑子

我采用的安装包：点击此处下载

几点约定：

  • 任何文件存放目录不出现中文、特殊符号，尽量只包含英文字母和下划线

    注意：我上传的安装包第二级目录中有个“位”字，如使用则自行删去

  • 任何可执行文件默认以管理员身份运行

  • 操作环境 Windows 10 1903 ，建议 Win10 早期版本或 Win7、xp 用户更新系统

  • 安装软件为 64 位版本，一般不推荐使用 32 位版本，如果你还在使用 32 位电脑，建议换机

友情提示：

  • 本文虽图文并茂，但并未将每个详细步骤都截图展示，请认真阅读文字部分，并注意观察屏幕上的各种提示语，有不认识的单词建议查词典，因为眼瞎而安装失败者自当反省

第一步 java环境配置

打开系统设置，选择“应用”

搜索 java ，然后卸载所有 java 运行环境程序

然后在安装包里找到并运行 jre-7u9-windows-64.exe 安装Java虚拟机

第二步 许可证修改匹配

右键“开始”按钮，选择“系统”，复制你的设备名称

用记事本打开许可证文件 NX8.5.lic ，将 SERVER 后的第一个字符串修改为你的计算机名

第三步 兼容性配置

设置 SPLMLicenseServer_v5.3.1.exe 文件属性，将兼容模式设置为 Windows7

同样地，也设置 Launch.exe

第四步 许可证的安装

打开安装程序 Launch.exe，并选择第二项安装许可证

然后点击“确定”

点击“下一步”直到“选择许可证文件”这步，然后选择刚刚修改过设备名称的许可证文件 NX8.5.lic

然后继续“下一步”，直到完成安装

第五步 主程序的安装

回到 launch 程序，选择第三项安装主程序

然后按照提示正常安装即可，可以根据喜好调整安装配置，不会调整默认亦可

注意在“许可”这步中，一定要确认文本框中显示的是你的设备名称，如不是，请从头再来

经过近乎绝望的漫长的安装过程（大约需要 100 年），完成安装程序

然后回到 launch ，点击 exit 退出

第六步 破解文件的覆盖

在安装包目录中找到这五个文件夹，复制他们

将他们覆盖粘贴到 ug nx 8.5 的安装目录中：

注意：默认安装目录为 C:\Program Files\Siemens\NX 8.5 如在安装过程中有修改，则自行寻找

然后打开第六个文件夹 UGS_Licensing_SSQ 将文件 ugslmd.exe 复制

覆盖粘贴到安装包目录中包含 lmtools.exe 这个文件的文件夹中

第七步 许可证服务器的配置

运行 lmtools.exe

在上方选择到 Config Services 选项卡

接着仔细看下图，认真按照教程操作

  1. 随意填写一个名字（绝不可有中文）
  2. 选择与 lmtools.exe 同文件夹的 lmgrd.exe
  3. 选择与 lmtools.exe 同文件夹的 NX8.5.lic
  4. 选择与 lmtools.exe 同文件夹的 ugslicensing.log
  5. 勾上
  6. 勾上
  7. 点击

然后在下述弹窗点击“是”

然后切换到 Start/Stop/Reread 选项卡

观察到已经选中刚刚配置的许可证服务器名称，点击 Stop Server

稍等数秒，确认在左下角看到 Stopping Server

然后点击 Start Server，并确认在左下角看到 Server Start Successful

此时许可证服务器配置完成，可关闭本程序

第八步 重启电脑

此时请务必重启你的电脑

第九步 Enjoy

在开始菜单中打开 Siemens NX 8.5 - Nx 8.5

Then enjoy it
