经过修改尝试不难得到：

  • 参数 ForegroundText 控制中号磁贴下方的 App 名称字体颜色，可选 Light 、 Dark 两个值
  • 参数 BackgroundColor 控制磁贴默认背景颜色，值为 16进制颜色代码 或 常用颜色名
  • 参数 ShowNameOnSquare150x150Logo 控制中号磁贴下方是否显示 App 名称，可选 on 、 off 两个值
  • 参数 Square150x150Logo 控制中号磁贴显示的图标，值为 图片路径
  • 参数 Square70x70Logo 控制小号磁贴显示的图标，值为 图片路径

微软开发者文档

本着不得其源不罢休的精神，我在 Microsoft Docs 中找到了这样一篇 Doc：

如何自定义桌面应用的“开始”屏幕磁贴（Windows 运行时应用）

看完官方文档，还有以下几点值得学习的地方：

  1. 配置文件中的 Square150x150Logo 和 Square70x70Logo 属性是可选的，但所有其他属性都是必需的。（参考 Origin 的配置方案）

  2. 颜色预定义字符串有：

颜色名 预定义字符串
黑色 black
银色 sliver
灰色 gray
白色 white
栗色 maroon
红色 red
紫色 purple
紫红色 fuchsia
绿色 green
石灰色 lime
橄榄色 olive
黄色 yellow
藏青色 navy
蓝色 blue
凫蓝色 teal
浅绿色 aqua

  1. 对磁贴图像的限制：

    • 尺寸小于等于 1024x1024 像素
    • 文件大小小于或等于 200 KB
    • 文件类型为 .png、.jpg、.jpeg 或 .gif

  2. 如果你选择自定义图像，则必须针对 Square150x150Logo 和 Square70x70Logo 属性指定图像。如果仅指定其中一个属性，则忽略整个 XML 文件，并将默认样式（应用图标和背景）应用到磁贴。

  3. 一个优秀的开发者应当指定呈比例的、本地化的和高对比度的图像，详见 Doc 步骤 6
